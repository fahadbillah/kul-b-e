kul-b-e Homework
==================================

Getting Started
---------------

```sh
# clone it
git clone git@github.com:fahadbillah/kul-b-e.git
cd kul-b-e

# Install dependencies
npm install

# Start development live-reload server
PORT=3000 npm run dev

# Start production server:
PORT=3000 npm start
```
